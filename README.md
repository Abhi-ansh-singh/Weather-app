## Run Locally

Clone the project

```bash
  git clone https://github.com/Abhi-ansh-singh/Weather-app.git
```

Go to the project directory

```bash
  cd WeatherApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
